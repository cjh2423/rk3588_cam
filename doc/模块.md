# 模块功能说明与进度

## 1. 已实现模块

### 1.1 App 层 (业务逻辑)
- **AppController**: 核心调度器，管理线程生命周期。
- **PreprocessingThread**: 集成 RGA 硬件加速，支持 Letterbox 预处理。
- **InferenceThread**: 异步推理引擎，负责检测逻辑。
- **PerformanceMonitor**: FPS 统计与性能监控。

### 1.2 Core 层 (算法/库适配)
- **ModelManager**: 统一管理 YOLOv8 和 FaceNet 模型的加载。
- **YOLOv8-face**: 适配 RK3588 NPU 的人脸检测实现。
- **FaceNet**: 特征提取模型适配（待进一步集成到业务流）。
- **Postprocess**: 结果解析与坐标还原算法。

### 1.3 Hardware 层
- **CameraDevice**: 基于 V4L2 的异步视频流采集。
